# frontend-journey

**The rule I follow**: every concept I learn is applied the same day to something of my own. This repo is the proof, commit by commit.

## Structure

```
/html-css        → layouts, responsive design, components
/javascript      → fundamentals: functions, arrays, objects, DOM
/async-apis      → promises, fetch, REST APIs
/react           → components and mini-apps
/experiments     → small personal builds
```

*(Structure grows as I progress — restarted June 2026.)*
