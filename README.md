# learning-js
 
My JavaScript & React learning journal — daily exercises from the Codecademy **Frontend Engineer** path, plus small personal experiments.
 
**The rule I follow**: every concept I learn is applied the same day to something of my own. This repo is the proof, commit by commit.
 
## Structure
 
```
/fundamentals    → JS basics: functions, arrays, objects, loops
/dom             → DOM manipulation exercises
/async           → promises, fetch, APIs
/react           → React components and mini-apps
/experiments     → small personal builds
```
 
*(Structure grows as I progress — started June 2026.)*
